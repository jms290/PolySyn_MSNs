README file for the stat_maps.csv file

All maps are in MNI space aligned to a mid-resolution surface mesh (N=40962 vertices per hemisphere; commonly used in CIVET)

500parc = parcellation used in many papers from the Bullmore lab, it's a sub-parcellation of the Desikan-Killiany atlas (https://pubmed.ncbi.nlm.nih.gov/22094643/)

glasser = parcellation from Matt Glasser's Nature paper, also known as the "HCP" or "HCP_MMP" parcellation (https://www.ncbi.nlm.nih.gov/pmc/articles/PMC4990127/)

vonEconomo = 7 class parcellation based on von Economo and Koskinas 1927 (used here: https://pubmed.ncbi.nlm.nih.gov/29853553/)

yeo7 = Thomas Yeo's canonical functional network parcellation (N=7; used here: https://pubmed.ncbi.nlm.nih.gov/29853553/)

yeo17 = Thomas Yeo's canonical functional network parcellation (N=17; used here: https://pubmed.ncbi.nlm.nih.gov/29853553/) 

HCPscaling = allometric surface area scaling map in the HCP dataset (https://pubmed.ncbi.nlm.nih.gov/29853553/)

NIHscaling = allometric surface area scaling map in the NIH dataset (https://pubmed.ncbi.nlm.nih.gov/29853553/)

PNCscaling = allometric surface area scaling map in the PNC dataset (https://pubmed.ncbi.nlm.nih.gov/29853553/)

glasser_GI = glycolytic index map (created from the next two maps), a proxy for aerobic glycolysis (https://www.ncbi.nlm.nih.gov/pubmed/20837536)

glasser_CMRO2 = cerebral metabolic rate of oxygen map (https://www.ncbi.nlm.nih.gov/pubmed/20837536)

glasser_CMRGlu = cerebral metabolic rate of glucose map (https://www.ncbi.nlm.nih.gov/pubmed/20837536)

glasser_CBV = cerebral blood volume map (https://www.ncbi.nlm.nih.gov/pubmed/20837536)

glasser_CBF = cerebral blood flow map (https://www.ncbi.nlm.nih.gov/pubmed/20837536)

hill.evo_remapped = evolutionary scaling (macaque to human) map (https://pubmed.ncbi.nlm.nih.gov/20624964)

hill.devo_remapped = developmental scaling (infant to adult) map (https://pubmed.ncbi.nlm.nih.gov/20624964)

asl = average arterial spin labeling map in the PNC cohort (https://pubmed.ncbi.nlm.nih.gov/24912164)

BBsupra = supragranular layer stain intensity/density map from BigBrain (https://pubmed.ncbi.nlm.nih.gov/31107870 and used here: https://pubmed.ncbi.nlm.nih.gov/32620757)

BBinfra = infragranular layer stain intensity/density map from BigBrain (https://pubmed.ncbi.nlm.nih.gov/31107870 and used here: https://pubmed.ncbi.nlm.nih.gov/32620757)
